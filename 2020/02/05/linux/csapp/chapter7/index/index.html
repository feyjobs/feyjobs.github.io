  • 加载时链接
  • 运行时

编译器

以gcc为例，自上而下其实运行的过程如下

  • cpp, 注意这个不是c++，而是一个预处理程序

  • as 一个汇编器, 把汇编语言转成机器语言
  • ld 连接器把各个文件链接起来
